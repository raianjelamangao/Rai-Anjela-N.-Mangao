# Calculator Application

## Setup

1. Install dependencies:

```bash
npm init -y
npm install express
```

2. Start server:

```bash
node server.js
```

3. Open `http://localhost:3000` in your browser.

## Files

- `index.html` - frontend UI and JS call to `/api/calc`
- `server.js` - Express server with calculation endpoint
- `readme.md` - this file
